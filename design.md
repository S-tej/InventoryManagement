# Design Overview

## Architecture
- Backend: Node.js + Express
- Database: MongoDB
- AI: Gemini API
- Fallback: Rule-based engine

## Key Features
- Batch-based inventory tracking
- FIFO consumption
- AI query system
- Fallback for reliability
- Cron-based prediction engine

## Data Flow
User → API → DB → AI/Fallback → Response

## Future Improvements
- Real-time notifications
- Mobile app
- Advanced ML forecasting
